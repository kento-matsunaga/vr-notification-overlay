# VRNotify

Windows の通知を VR 空間内にそのまま表示するアプリです。VR を外さずに Discord や LINE の通知を確認できます。

## できること

- VR ヘッドセットをかぶったまま、あらゆる Windows アプリの通知を確認
- 通知はヘッドセット上部に自動で表示され、数秒で消えます
- アプリごとに通知の表示/非表示を設定可能
- おやすみモードで一時的に通知をオフに
- **SteamVR 自動起動** — SteamVR を起動するだけで VRNotify も一緒に立ち上がります
- **VR 内設定パネル** — VR ダッシュボードから直接おやすみモード・透明度・表示時間を変更

## 動作環境

- **Windows 10 / 11**（64ビット）
- **SteamVR 対応の VR ヘッドセット**（Quest 2/3 の PC 接続、Valve Index、HTC VIVE など）

> PC に接続した状態で使います。Quest 単体（スタンドアロン）では動作しません。

## インストール

1. `VRNotify-1.0.0-Installer.exe` をダブルクリック
2. 「このアプリがデバイスに変更を加えることを許可しますか？」→ **はい**
3. インストール先を確認して **Install**
4. 完了を待つだけ！

> **「Windows によって PC が保護されました」と出たら**: 「詳細情報」→「実行」で進めてください。

## 使い方

### 起動方法（2つ）

| 方法 | 説明 |
|------|------|
| **SteamVR 自動起動**（おすすめ） | SteamVR を起動すると VRNotify も自動で起動します。初回のみスタートメニューから手動起動が必要です。 |
| **スタートメニュー** | スタートメニュー → **VRNotify** をクリック。 |

タスクバー右下に青い **VR** アイコンが出たら起動完了です。あとはバックグラウンドで動きます。

> **初回セットアップ**: インストール後、最初の1回だけスタートメニューから VRNotify を起動してください。これで SteamVR にアプリが登録され、次回以降は SteamVR と一緒に自動起動します。

### VR 内の通知

SteamVR を起動すると、通知が VR 空間のヘッドセット上部に自動表示されます。

- 通知はカード型で、アプリ名・タイトル・本文が表示されます
- デフォルト 5 秒で自動的に消えます
- SteamVR の起動順は自由です（先でも後でも OK）

### VR ダッシュボード設定パネル

VR 内から直接設定を変更できます。

1. コントローラーの **メニューボタン** を押して SteamVR ダッシュボードを開く
2. 下部のタブバーに **VRNotify 設定** タブが表示されます
3. レーザーポインターで操作：

| 設定 | 操作 |
|------|------|
| **おやすみモード** | 「オフ」「すべて非表示」「重要な通知のみ」の3ボタンから選択 |
| **不透明度** | スライダーで 10%〜100% を調整 |
| **表示時間** | スライダーで 2〜15 秒を調整 |
| **ステータス** | 接続状況を確認 |

### デスクトップ設定画面

トレイの **VR** アイコンをダブルクリック、または右クリック →「設定...」で設定画面を開きます。

| タブ | できること |
|------|----------|
| **General** | おやすみモード（全通知オフ / 高優先度のみ / オフ） |
| **Filter** | アプリごとに通知の表示/非表示を切り替え |
| **Display** | 表示秒数（2〜15秒）、透明度、サイズの調整 |
| **History** | 過去の通知を確認・削除 |

**フィルタの使い方例**:
- 「Discord と LINE だけ表示したい」→ Block all モードにして Discord と LINE を Allowed に
- 「特定アプリだけ止めたい」→ Allow all モードのままそのアプリを Blocked に

### おやすみモード

トレイアイコンを **右クリック** → **おやすみモード** から素早く切り替えできます。設定画面を開く必要はありません。

### 終了

トレイアイコンを **右クリック** → **終了**。

## アンインストール

Windows の **設定** → **アプリ** → **VRNotify** → **アンインストール**。

## うまくいかないとき

### 通知が VR に表示されない

1. SteamVR が起動しているか確認（起動後 5 秒ほど待つ）
2. Windows の通知が有効か確認（設定 → システム → 通知）
3. トレイに VR アイコンがあるか確認（なければスタートメニューから起動）

### SteamVR で自動起動しない

初回は手動でスタートメニューから VRNotify を起動してください。これで SteamVR にアプリが登録されます。次回以降は SteamVR と一緒に自動起動します。

### 起動時にエラーが出る

スタートメニューから起動してください。exe の直接実行では動作しません。
それでも解決しない場合はインストーラーを再実行してください。

### トレイアイコンが見つからない

タスクバー右端の **^** ボタンをクリックすると、隠れたアイコンが表示されます。
